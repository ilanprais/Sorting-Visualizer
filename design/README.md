# Sorting Visualizer Web Application

Simple webpage design using html and css. Webpage code using javascript.
website visuaizing *Sorting Algorithms*.

Website name - VisualSort

**Algorithms to support:**
- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort
- Quick Sort
- Heap Sort

- run controlls

  - pause
  - speed
  - reset
- algo controlls:
  - algo coice menu
- array controlls:
  - new array
  - array size
  - Reverse
  
**On Display Stats:**
- time complexity
- comparisons
