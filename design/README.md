# Sorting Visualizer Web Application

## Requirements

simple webpage design using html and css. webpage code using javascript.
website visuaizing *sorting algorithms*.

**Algorithms to support:**
- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort
- Quick Sort
- Heap Sort

**Controlls:**
- run controlls
  - play (run)
  - pause
  - speed
  - reset
- algo controlls:
  - algo coice menu
- array controlls:
  - new array
  - array size
  - Reverse
